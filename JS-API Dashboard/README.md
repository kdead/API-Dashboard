# Daily Dashboard

This project is a JavaScript API dashboard that brings together helpful tools, information, and entertainment in one place.

## Features

- Weather search by city
- GitHub profile search
- Joke generator
- Advice/quote generator
- Fun fact generator
- Radio station search
- Dev news loader
- Food inspiration section

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API
- Async/Await

## APIs Used

- Open-Meteo
- GitHub API
- JokeAPI
- Advice Slip API
- Useless Facts API
- Radio Browser API
- Hashnode GraphQL API
- Foodish API

## How to Use

1. Open the project in your browser.
2. Use the search fields and buttons in each card.
3. View dynamic results loaded from APIs.

## Project Goal

The goal of this project was to practice working with public APIs, asynchronous JavaScript, and DOM manipulation by building a dashboard with multiple mini apps.
